# live4api

LIVE4 Public API

With Maven:

```
<dependency>
    <groupId>io.live4</groupId>
    <artifactId>live4api3</artifactId>
    <version>3.0.2</version>
</dependency>
```

With Gradle:
```
repositories {
    maven {
        url "http://kote.videogorillas.com/m2repo/"
    }
}

dependencies {
    compile 'io.live4:live4api3:3.0.2'
}
```
